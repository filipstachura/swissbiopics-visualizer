Using SwissBioPics in your own site

Using SwissBioPics requires two templates. One for styling and one for how the side bar is supposed to render. 

      <template id="sibSwissBioPicsStyle">
       <style>
	ul > li > a {
	  font-style:oblique;
	}
	ul.notpresent {
	  display:none;
	}
      </style>
    </template>
    <template id="sibSwissBioPicsSlLiItem">
      <li class="subcellular_location">
	<a class="subcell_name"></a> <-- the class name is required and textContent will be set on it
	<span class="subcell_description"></span> <-- the class name is required and textContent will be set on it
      </li>
    </template>
Then use 
<sib-swissbiopics-sl taxid="A NCBI taxid" sls="Comma seperated list of SubcellularLocation identifiers"></sib-swissbiopics-sl>
Make sure to import the module via node or use the latest via www.swissbiopics.org 
        <script type="module" src="https://www.swissbiopics.org/static/swissbiopics.js"></script>
        <script defer>
            if (! window.customElements.get("sib-swissbiopics-sl"))
            window.customElements.define("sib-swissbiopics-sl", SwissBioPicsSL);
        </script>

            <style>
            svg text {
                fill:unset;
                display:none;
            }
            </style>
        </template>
